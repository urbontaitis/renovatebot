# Renovatebot presentation

[Renovatebot demo - keep your dependencies up to date](renovate-bot.md)

Renovatebot demo using bitbucket server

## Install renovate

```bash
npm i -g renovate
```

## Update config.json:
- username
- password (Personal access token from bitbucket)
- gitAuthor
- repositories

## Run renovate bot:
```bash
> renovate
```
